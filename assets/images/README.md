# Graduation Images for Hero Carousel

## Required Images

To display the hero carousel properly, please add the following graduation images to this folder:

1. **graduation1.jpg** - Main graduation ceremony image (students in caps and gowns)
2. **graduation2.jpg** - Alternative graduation image (celebration scene)
3. **graduation3.jpg** - Third graduation image (academic achievement)

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 pixels (Full HD)
- **Aspect Ratio**: 16:9
- **Maximum File Size**: 2MB per image (for optimal loading speed)
- **Quality**: High resolution for crisp display

## How to Add Images

1. Save your graduation photos in this folder (`assets/images/`)
2. Name them exactly as: `graduation1.jpg`, `graduation2.jpg`, `graduation3.jpg`
3. Refresh your browser to see the images in the carousel

## Note

The carousel will display a gradient background fallback until images are added. Once images are in place, they will automatically appear with smooth fade and zoom animations.
